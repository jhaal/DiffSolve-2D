## Features

### Any Core Size

Any 2-D Core size input will work. Modify in the header file

### Compile

Use makefile


```shell
$ make -k makefile
```
or, at the terminal

Call

```shell
$ g++ DiffSolve-2D.cpp -o DiffSolve-2D && ./DiffSolve-2D
```
or,

```shell
$ c++ DiffSolve-2D.cpp -o DiffSolve-2D && ./DiffSolve-2D
```

### Plotting Of Results

```shell
$ gnuplot < plot.gp
```
